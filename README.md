# yfang-utils

圆方软件--2--前端方法库。

提供齐全，兼容性良好,测试全面的方法供日常使用。

## 安装

```
npm install yfang-utils --save
```

## 使用

```ts
import { deepClone } from 'yfang-utils'

const data = deepClone({name: 'yfang-utils'})
```
